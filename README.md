# WebStudio – HTML & CSS

This project is the second homework from the GoIT HTML/CSS course.  
It contains HTML markup and styling for the WebStudio landing page using CSS.

## 🧰 Technologies

- HTML5
- CSS3
- Git
- GitHub

## 🚀 Features

- Semantic HTML structure
- CSS styling and layout
- Responsive design
- Navigation menu
- Sections: Hero, Features, Team, Portfolio, Footer

## 📦 How to use

Open the `index.html` file in your browser to view the project.
